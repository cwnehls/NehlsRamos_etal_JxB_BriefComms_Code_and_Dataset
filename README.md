# NehlsRamos_etal_2025
Code and Dataset for "Rubisco Dark Inhibition in Angiosperms Shows a Complex Distribution"

Authors: Connor Nehls-Ramos, Elizabete Carmo-Silva, Douglas J. Orr

The R code supplied in this folder is for the analyses and production of plots for the dataset used in the paper stated above.
All code included is intended to be run individually for each file, and run at once within each file-- unless specified otherwise
in the description included in the file. The dataset and code includes all necessary information to recreate the figures in the paper.

Code file names are presented in connection to the figure or section of paper relevant for said code. The "S" signifies Supplemental
